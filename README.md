

Aplicación Móvil — Academia SmartCell
Descripción

-Academia SmartCell es una aplicación móvil desarrollada en Kotlin utilizando el entorno de desarrollo Android Studio. La aplicación tiene como propósito mostrar información general de una academia, incluyendo sus cursos, profesores y detalles institucionales.

-El proyecto fue diseñado para ser compatible con versiones bajas de Android, utilizando archivos XML para el diseño de interfaces y archivos .kt para la lógica de la aplicación.

-Este repositorio representa distintas etapas del desarrollo de la aplicación, desde un prototipo inicial hasta versiones más dinámicas e informativas.

Tecnologías Utilizadas
-Lenguaje de programación: Kotlin
-IDE de desarrollo: Android Studio
-Diseño de interfaces: XML
-Control de versiones: Git y GitHub
-Compatibilidad: Versiones bajas de Android

-Estructura de Ramas

El repositorio contiene tres ramas principales:

*main
  Contiene el primer prototipo funcional de la aplicación, incluyendo la estructura inicial del proyecto y las primeras interfaces.

*dblocal
  En esta rama se implementó una base de datos local, con pruebas de almacenamiento de información por persona dentro de la aplicación.

*dinamico
  En esta rama se desarrolló una versión informativa de la aplicación, donde se muestran detalles de la academia, cursos, profesores y otros elementos institucionales.

Requisitos

Para ejecutar el proyecto se recomienda contar con:

*Android Studio instalado
*JDK configurado correctamente
*Conexión a Internet para descargar dependencias
*SDK de Android compatible con versiones bajas de Android
*Cómo Clonar y Ejecutar el Proyecto en Android Studio
*Abrir Android Studio
  --Seleccionar la opción:
   --Get from VCS

     -Copiar la URL del repositorio y pegarla en el campo correspondiente:
      https://github.com/tu-usuario/tu-repositorio.git

*Seleccionar la carpeta donde se descargará el proyecto
*Esperar a que Android Studio sincronice el proyecto automáticamente
*Ejecutar la aplicación en un emulador o dispositivo físico
Recomendaciones Importantes

Para evitar problemas con dependencias y configuraciones, se recomienda:

>Permitir que Gradle descargue automáticamente las dependencias
>No modificar manualmente los archivos de configuración iniciales del proyecto
>Utilizar la misma versión de Android Studio o una versión compatible
>Verificar que el SDK requerido esté instalado antes de ejecutar la aplicación
Ejecutar la opción:
>Sync Project with Gradle Files

(si ocurre algún error al abrir el proyecto)
-Estas prácticas ayudan a garantizar que el proyecto se ejecute correctamente en diferentes entornos de desarrollo.

Estado del Proyecto

El proyecto se encuentra en desarrollo y representa un proceso de aprendizaje en el desarrollo de aplicaciones móviles con Kotlin y Android Studio.
